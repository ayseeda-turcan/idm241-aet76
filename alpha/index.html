Pinterest Hover Preview

Triggers

  • Hover On
  • Hover Off

Rules

  1. Hover On
    • A semi-transparent dark overlay appears over the image
    • "Share" button and "More Options" icons appear
    • Cursor changes to pointer
  2. Hover Off
    • Overlay disappears
    • Icons disappear
    • Pointer returns to default cursor

Loops
