Rules

Pinterest Card

  1. Hover On
    • A semi-transparent dark overlay appears over the image, at 45% opacity
    • "Share" button and "More Options" icons appear, 30px by 30px
    • "Open" text appears, font size 20px
    • Cursor changes to pointer
  2. Hover Off
    • Overlay disappears
    • Icons disappear
    • "Open" text disappears
    • Pointer returns to default cursor
