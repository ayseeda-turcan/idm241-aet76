Rules

Overlay

  1. Hover On
    • A semi-transparent dark radial gradient fades into view, along with an "Open" sign
    • "Share" button and "More Options" icons slide up from the bottom right of the image
    • Cursor changes to pointer
  2. Hover Off
    • Overlay and "Open" fade out slowly
    • Icons slide down from the bottom right of the image
    • Pointer returns to default cursor

Share

  1. Hover On
    • Button transforms into a slightly bigger shape
    • Button background colour gets slightly darker
  2. Hover Off
    • Button transforms back into its original, smaller shape
    • Button background returns back to its lighter colour
  3. Click
    • Button transforms into a slightly smaller shape, compared the the hover on state
    • After the click, button shape changes into a larger shape, revealing share options and icons
